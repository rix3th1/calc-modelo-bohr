# Calculadora de Modelo de Bohr

## Requisitos

Para ejecutar este programa, necesitarás instalar algunos paquetes adicionales. Puedes hacerlo utilizando el siguiente comando:

```bash
pip install -r requirements.txt
```

## Ejecución

Para ejecutar este programa, simplemente ejecuta el siguiente comando:

```bash
python main.py
```

Esto abrirá una ventana gráfica con el programa.

## Compilación

Para compilar este programa, puedes utilizar el siguiente comando:

```bash
pyinstaller --onefile --windowed main.py --name calc-modelo-bohr
```

Esto creará un archivo ejecutable llamado `calc-modelo-bohr.exe` en la carpeta /dist/.
